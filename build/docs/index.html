Cisco UI Kit

Cisco user interface kit 1.2.2

A web-based implementation of the UI design language sponsored by Cisco Brand

  •   Intro  
  • Download
  • Change Log

Build Amazing Apps!

The Cisco UI kit is designed to be the modern, lightweight presentation layer you need to quickly skin your components with the latest from Cisco Brand. The kit comes bundled with over 30 common patterns which can be used as building blocks with which you can construct your application.

Using React or Angular? Great! This kit is javascript-agnostic so use whatever javascript tooling that works for you and your team.

The kit itself is internal open source so anyone can contribute. We welcome both designers and developers and believe that this kit is an important collaboration keystone between those two groups.

Browser Support

The Cisco UI Kit is geared for modern browsers only. Minimum browser versions include:
Chrome 43 Edge 12 Firefox 41 Internet Explorer 11 Opera 33 Safari 10

Coming Soon

Angular web components. We have developers busy working on building out a set of reusable Angular 1.x and Angular 4.x components which follow the Cisco UI kit spec. You can access their code via the Git repositories here and here.

Angular sandbox. In addition to the Angular web components we are also planning on making the Angular Sandbox available. This sandbox came from the Apollo team (thanks!) and will provide working component examples with documentation - very cool.

React web components. Another team has been busy building out a set of React components for the UI kit. You can find the code in Git here.

Once the new digital web standards are available from Cisco Brand we will get busy working on version 2.0 of the UI kit which will look a lot more like the cisco.com home page. Stay tuned - we will announce once we get our grubby little hands on the spec.
